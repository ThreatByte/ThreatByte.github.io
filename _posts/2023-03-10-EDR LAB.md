## OPEN EDR Steps
![image](https://user-images.githubusercontent.com/122228333/224412299-63410352-fc7b-4689-803a-45ffaad8b538.png)
- Assets Device to Confirm a device(Workstation) is installed with Endpoint Client.

![image](https://user-images.githubusercontent.com/122228333/224412877-a62048c8-2d80-4ef6-b0cc-e35c8cd2d644.png)
- Click Install Additional Xcitium Packages click option "Install Xcitium Client - EDR"

- You Should see a prompt that states the endpoint will be restarting in 5 minutes. If you have not received one change the network type from Host only back to Network Address Translation.

![image](https://user-images.githubusercontent.com/122228333/224416574-d2c8be26-df94-4e26-9f6f-f8884d2b177f.png)
- Verify Endpoint Detection Response is installed on the client.

## OPEN EDR Functionality

![image](https://user-images.githubusercontent.com/122228333/224417448-87bb4dcc-b8be-47d6-933c-43674a26b9e9.png)

- Service:
 - Events Processed(Events that are occuring on Client Workstation)
 - Investigated Alerts(Alerts that are being Investigated by SOC Team)
 - Incidents(Malicious activity occuring on Workstation)
 - Tickets Notified(Number of Tickets Closed Security Analyst Team)
 - Unread Reports(Reports generated by workstation that need to be read still)
 
 ![image](https://user-images.githubusercontent.com/122228333/224418244-d45655bf-e968-4fea-8072-d71a690b9758.png)

- Endpoint:
 - Provie Information about Alerts, Malware Detected, Devices that are connected to Xicitum Enterprise Platform, Contained Files(Files that are malicious).

![image](https://user-images.githubusercontent.com/122228333/224418632-62a7abff-611c-4404-abb8-961d859ddc77.png)

- Audit:
 - Devices that are connected and can be audited.

![image](https://user-images.githubusercontent.com/122228333/224418892-f843bc80-f9da-451f-944f-814e043547a2.png)

- Compliance:
 - Determine if a device has virus, needs to be scanned, black listed applications, etc.
 
![image](https://user-images.githubusercontent.com/122228333/224419374-9c621f65-27da-4091-a740-b2ed6d1e1ee1.png)

- Xcitium Verdit Cloud:
 - Determine through Threat Intelligence what peice of malware could be on Workstation.
 
 ![image](https://user-images.githubusercontent.com/122228333/224420012-29c76dfe-6e63-4145-a040-e83650ec42de.png)

- Reports:
 - Generate Reports about Workstation.

![image](https://user-images.githubusercontent.com/122228333/224420262-912053ec-b0ea-4266-8f14-f67578e4e9ef.png)

- Notifications:
 - Pending Notifications about Workstation.

![image](https://user-images.githubusercontent.com/122228333/224420504-67d30a64-0b9e-450f-bb6c-74c8a17d7356.png)

- Audit Logs:
 - Logs that can be auditied.
