## OPEN EDR Steps
![image](https://user-images.githubusercontent.com/122228333/224412299-63410352-fc7b-4689-803a-45ffaad8b538.png)

- Assets Device to Confirm a device(Workstation) is installed with Endpoint Client.

![image](https://user-images.githubusercontent.com/122228333/224412877-a62048c8-2d80-4ef6-b0cc-e35c8cd2d644.png)

- Click Install Additional Xcitium Packages click option "Install Xcitium Client - EDR"

- You Should see a prompt that states the endpoint will be restarting in 5 minutes. The Workstation must be on a Nat network to communicate with OpenEDR.

![image](https://user-images.githubusercontent.com/122228333/224416574-d2c8be26-df94-4e26-9f6f-f8884d2b177f.png)

- Verify Endpoint Detection Response is installed on the client.

![image](https://user-images.githubusercontent.com/122228333/224430744-e87be4f3-d35b-4a9e-827c-23acf69b9502.png)

- Install Security also to receive more functionality like Antivirus, Firewall, etc.

![image](https://user-images.githubusercontent.com/122228333/224432198-142b608a-28a1-42aa-89ef-91c7d73eb0aa.png)

![image](https://user-images.githubusercontent.com/122228333/224432949-0bc3b340-3e62-42d2-8301-abaff271300d.png)

- What you get when you install Security addon is Commodo AV, and share space.

![image](https://user-images.githubusercontent.com/122228333/224433341-cd387798-6f72-4990-bfe4-9bf7437a0728.png)

- AV, Firewall, HIPS(Host Intrusion Prevention), Containment, Data Loss Prevention, File Rating, Advanced Protection, Website Filtering.


## OPEN EDR Functionality

![image](https://user-images.githubusercontent.com/122228333/224417448-87bb4dcc-b8be-47d6-933c-43674a26b9e9.png)

- Service:
 - Events Processed(Events that are occuring on Client Workstation)
 - Investigated Alerts(Alerts that are being Investigated by SOC Team)
 - Incidents(Malicious activity occuring on Workstation)
 - Tickets Notified(Number of Tickets Closed Security Analyst Team)
 - Unread Reports(Reports generated by workstation that need to be read still)
 
 ![image](https://user-images.githubusercontent.com/122228333/224418244-d45655bf-e968-4fea-8072-d71a690b9758.png)

- Endpoint:
 - Provie Information about Alerts, Malware Detected, Devices that are connected to Xicitum Enterprise Platform, Contained Files(Files that are malicious).

![image](https://user-images.githubusercontent.com/122228333/224418632-62a7abff-611c-4404-abb8-961d859ddc77.png)

- Audit:
 - Devices that are connected and can be audited.

![image](https://user-images.githubusercontent.com/122228333/224418892-f843bc80-f9da-451f-944f-814e043547a2.png)

- Compliance:
 - Determine if a device has virus, needs to be scanned, black listed applications, etc.
 
![image](https://user-images.githubusercontent.com/122228333/224419374-9c621f65-27da-4091-a740-b2ed6d1e1ee1.png)

- Xcitium Verdit Cloud:
 - Determine through Threat Intelligence what peice of malware could be on Workstation.
 
 ![image](https://user-images.githubusercontent.com/122228333/224420012-29c76dfe-6e63-4145-a040-e83650ec42de.png)

- Reports:
 - Generate Reports about Workstation.

![image](https://user-images.githubusercontent.com/122228333/224420262-912053ec-b0ea-4266-8f14-f67578e4e9ef.png)

- Notifications:
 - Pending Notifications about Workstation.

![image](https://user-images.githubusercontent.com/122228333/224420504-67d30a64-0b9e-450f-bb6c-74c8a17d7356.png)

- Audit Logs:
 - Logs that can be auditied.

![image](https://user-images.githubusercontent.com/122228333/224433891-ca60c4eb-0224-4631-aab8-df50b1d04358.png)

![image](https://user-images.githubusercontent.com/122228333/224434048-acaf5407-aadf-4fd7-b6ce-ea9aa0d8dfef.png)

- Alerts:
 - Events, Child Processes, Device Names, Event Time Stamp, Mitre Tactic Alerts.

![image](https://user-images.githubusercontent.com/122228333/224434349-2dfb1029-bb09-4f2b-9427-ca7fec43e6c6.png)

- Investigate:
 - Processes that have been Written, Deleted, Query, etc.
 
![image](https://user-images.githubusercontent.com/122228333/224434497-0d94d142-e610-4683-960b-3b8c4de985a1.png)

- Security Status:
 - Determine Endpoint Security Who is logged in, Is AV Updated, Version, Date Scanned, Malware Status.

![image](https://user-images.githubusercontent.com/122228333/224434663-a546279f-0c39-489f-ba12-9e34f78d1209.png)

- Blocked Threats:
 - Threats that have been blocked.

![image](https://user-images.githubusercontent.com/122228333/224434787-6b6bf8ae-37ff-4577-8508-6ff3692ad8d7.png)

- Quarintend Threats:
 - Threats that have been Quarintined.

![image](https://user-images.githubusercontent.com/122228333/224434920-98045ad0-c05f-433c-a6cd-0a627285af63.png)

- Contained Threats:
 - File Name, Path, Hashes, etc.

![image](https://user-images.githubusercontent.com/122228333/224435105-e2eb6f50-a1c1-43c4-8aa2-8421d4d2c862.png)

- HIDS:
 - Host Intrustion Detection System.

![image](https://user-images.githubusercontent.com/122228333/224435186-b2155156-cf8e-4afd-9e2e-72978cdc1690.png)

- Firewall Events:
 - Application, Source IP, Source Port, Destination IP, Destination Port

![image](https://user-images.githubusercontent.com/122228333/224435305-997f939c-edb1-4946-9d08-7abe79295dd8.png)

- Auto Run:
 - Applications that run by default.

![image](https://user-images.githubusercontent.com/122228333/224435394-48da1810-09a2-4573-b065-b05a628e4c47.png)

- File Rating:
 - Rating of a file. 

![image](https://user-images.githubusercontent.com/122228333/224435480-9ed42d28-ccaf-4cda-970e-2a4a37575048.png)

- Device Control:
 - Devices that are connected to the workstation.

![image](https://user-images.githubusercontent.com/122228333/224435582-cab09308-b169-45a4-9c30-91939282b23e.png)

- Data Loss Prevention

![image](https://user-images.githubusercontent.com/122228333/224435818-e8565f21-9870-4eb3-92f6-7c28126b2ac3.png)

Software Inventory:
 - Allow you to push patches to Workstation to download.

- There is a lot more features that End Point Detection and Response can offer these are just some notable ones.
